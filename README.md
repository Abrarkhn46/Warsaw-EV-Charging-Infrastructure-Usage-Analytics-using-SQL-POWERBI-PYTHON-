# Warsaw-EV-Charging-Infrastructure-Usage-Analytics-using-SQL-POWERBI-PYTHON-
Project Overview

This project analyzes Electric Vehicle (EV) charging infrastructure and usage patterns in Warsaw. The objective is to evaluate infrastructure readiness, identify consumption trends, segment user behavior, and generate data-driven insights to support strategic decision-making in EV ecosystem planning.

🎯 Key Objectives

Analyze total charging sessions, energy consumption, and revenue

Rank districts based on infrastructure readiness

Compare actual charging usage vs projected demand

Identify high-performing and underutilized districts

Segment charging behavior using clustering techniques

🛠 Tools & Technologies

SQL – Data extraction, aggregation, ranking, and district-level analysis

Python – Data preprocessing, EDA, clustering (K-Means), feature engineering

Power BI – Interactive dashboards and business insights visualization

📊 Key Analysis Performed

District-level energy consumption and revenue analysis

Infrastructure performance benchmarking

Charging session trend analysis

Cluster profiling based on:

Total sessions

Total energy consumed

Total spending

Average energy per session

Average spending per session

🔍 Business Insights

Identified districts with high energy demand but limited infrastructure.

Segmented EV users into behavioral clusters for targeted infrastructure planning.

Highlighted revenue-driving regions for strategic expansion.

Compared infrastructure readiness against actual utilization patterns.

📈 Outcome

The project provides actionable insights for policymakers, urban planners, and EV infrastructure companies to optimize charging station deployment and improve operational efficiency.
